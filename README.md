###Aprendendo a formatar um arquivo do tipo README 

Etapas de criação do repositório local:

mkdir git
cd git
Etapas de criação do repositório remoto:

Entrei no Github em minha conta e cliquei em Create new > New repository
Repository name: aula-git. Tipo: Public
Create repository
Clonando repositório remoto no local:

git clone https://github.com/fabivelosa/luiza_code.git
cd aula-git
touch README.md
git add README.md
git commit -m "Primeiro commit"
git push origin master
